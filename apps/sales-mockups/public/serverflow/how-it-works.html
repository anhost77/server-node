Under the Hood

See how ServerFlow transforms your idle VPS into a production-ready PaaS.

01

The Handshake

It starts with a single command. You paste it into your server terminal. This downloads our lightweight Golang binary.

The agent generates an Ed25519 keypair locally. It sends the public key to ServerFlow and keeps the private key on your machine. We never initiate connections; your server dials out to us via a secure persistent websocket.

02

The Build

When you push code or ask an AI agent to deploy, ServerFlow receives a secure webhook.

We instruct your server to pull the latest code. The build happens on your server (or you can use our cloud builders if you prefer). This means your source code never leaves your controlled environment if you don't want it to.

03

The Switch

Once the new version is healthy, our agent updates the Nginx configuration to point to the new port.

It's a zero-downtime switch. The old container is kept alive for 30 seconds in case a rollback is needed, then gracefully terminated. SSL certificates are auto-renewed by the agent.
